Kristine Gay M. Salinas - Personal Portfolio
--------------------------------------------

This is my final project for System Integration and Architecture. 

Sections:
- Home/Dashboard: Welcome section with my name, tagline, and social links.
- Projects: Shows my featured works with title, description, and link.
- Skills: Lists my technical skills.
- Talents: Personal strengths and creative talents.
- Organizations: Groups or clubs I am part of.
- Services: What I offer as an IT student.
- Testimonials: Quotes from co-workers, instructors or mentors.
- Contact: Email and location with contact form(no backend).

How to Run the Project:
-----------------------
1. Open any IDE that python compatible and Install Python (if not already installed).
2. Navigate to the `portfolio/` directory in your terminal or command prompt.
3. Run the local server:
   > python server.py
4. Open a browser and go to `http://localhost:8000`
